# Sorting Algorithms Visualizer

## pygame-sort-viz

Authors:
**Tim Ruscica** & **James Torkornoo**

- Currently supports:
  - Insertion Sort
  - Bubble Sort
  - Heap Sort
  - Shell Sort
  
&nbsp;

- Ensure python3 and pygame is installed.

- Run

```zsh
  python/python3 sort_viz.py 

```
